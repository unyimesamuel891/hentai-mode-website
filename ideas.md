## Design Approach Selected: Cyberpunk Anime Rebellion

### Design Movement
**Cyberpunk Anime Aesthetic** - A high-energy blend of anime character styling, neon colors, dystopian vibes, and chaotic digital energy. Think dark backgrounds with vibrant accent colors, glitch effects, and bold typography that screams rebellion and chaos.

### Core Principles
1. **High Contrast Drama** - Deep blacks and purples paired with electric neon pinks, cyans, and lime greens
2. **Anime-Inspired Visuals** - Anime character artwork, manga-style illustrations, and digital art aesthetics
3. **Chaotic Energy** - Asymmetric layouts, diagonal lines, glitch effects, and layered compositions
4. **Digital Rebellion** - Futuristic typography, tech-forward UI elements, and a sense of controlled chaos

### Color Philosophy
- **Primary Palette:** Deep purple (#1a0033) and black (#0a0a0a) as the foundation
- **Accent Colors:** Electric pink (#ff00ff), neon cyan (#00ffff), lime green (#00ff00)
- **Emotional Intent:** Edgy, rebellious, energetic, and unapologetically bold—perfect for a memecoin with "Hentai Mode" branding
- **Reasoning:** The dark base creates a premium feel while neon accents convey energy, chaos, and digital rebellion

### Layout Paradigm
- **Hero Section:** Full-width with anime character artwork, diagonal cuts, and glitch overlays
- **Asymmetric Grid:** Use diagonal dividers, offset sections, and layered cards to break monotony
- **Floating Elements:** Animated floating particles, glitch text, and layered depth
- **Vertical Flow:** Hero → About → Tokenomics → Social/Links with smooth transitions

### Signature Elements
1. **Glitch Text Effect** - Key headings with digital glitch animations
2. **Neon Glow Accents** - Glowing borders, shadows, and text highlights in pink/cyan
3. **Diagonal Section Dividers** - SVG wave/diagonal cuts between sections with clip-path
4. **Anime Character Showcase** - Central anime artwork with animated hover effects

### Interaction Philosophy
- **Responsive Hover States:** Buttons and links glow and scale on hover
- **Smooth Scroll Animations:** Elements fade and slide in as user scrolls
- **Glitch on Interaction:** Subtle glitch effects on button clicks and hovers
- **Particle Effects:** Background particles or floating elements that respond to scroll

### Animation Guidelines
- **Entrance Animations:** Stagger elements with 50-100ms delays, fade + slide-up (300ms ease-out)
- **Hover Effects:** Scale 1.05, glow intensity increase (150ms ease-out)
- **Glitch Effect:** Rapid color shifts and slight position jumps (100-200ms)
- **Scroll Animations:** Parallax on hero, fade-in on sections (200-400ms)
- **Micro-interactions:** Button press scale (0.95), smooth transitions on all interactive elements

### Typography System
- **Display Font:** "Space Mono" or "Courier Prime" (monospace for tech feel) - Bold, all-caps for headings
- **Body Font:** "Poppins" or "Roboto" (clean, modern sans-serif) - Regular weight for readability
- **Accent Font:** "Press Start 2P" or similar bitmap font for ultra-bold callouts (sparingly)
- **Hierarchy:** 
  - H1: 48-56px, Space Mono Bold, all-caps, with glitch effect
  - H2: 32-40px, Space Mono Bold, uppercase
  - Body: 16px, Poppins Regular, smooth and readable
  - Accent: 14-18px, Poppins Bold, neon-colored

## Implementation Notes
- Use Tailwind CSS with custom color variables for the neon palette
- Implement CSS animations for glitch effects and smooth transitions
- Leverage anime character artwork from the Twitter profile as hero imagery
- Create SVG dividers with diagonal/wave patterns for section breaks
- Use framer-motion for complex animations and scroll effects
- Ensure dark theme is default with high contrast for accessibility
